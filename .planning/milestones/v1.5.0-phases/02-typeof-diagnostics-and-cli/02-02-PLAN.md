Create `IoCTools.Generator.Tests/TypeOfRegistrationTests.cs` following the pattern in `ManualRegistrationOverlapTests.cs`. The test class should be `public sealed class TypeOfRegistrationTests`.

Use `#region` directives to organize tests by diagnostic code.

**Required test cases:**

`#region IOC090 - typeof() could use IoCTools attributes`
1. **AddScoped_TypeOf_NoAttributes_EmitsIOC090**: Source has `services.AddScoped(typeof(IMyService), typeof(MyServiceImpl))` where `MyServiceImpl` has NO IoCTools attributes. Expect IOC090.
2. **AddSingleton_TypeOf_NoAttributes_EmitsIOC090**: Same with AddSingleton. Expect IOC090.
3. **AddTransient_TypeOf_NoAttributes_EmitsIOC090**: Same with AddTransient. Expect IOC090.

`#region IOC091 - typeof() duplicates IoCTools`
4. **AddScoped_TypeOf_SameLifetime_EmitsIOC091**: `MyServiceImpl` has `[Scoped]` attribute AND `services.AddScoped(typeof(IMyService), typeof(MyServiceImpl))`. Expect IOC091.
5. **AddSingleton_TypeOf_SameLifetime_EmitsIOC091**: `[Singleton]` + `services.AddSingleton(typeof(...))`. Expect IOC091.

`#region IOC092 - typeof() lifetime mismatch`
6. **AddTransient_TypeOf_ScopedService_EmitsIOC092**: `MyServiceImpl` has `[Scoped]` but registered as `services.AddTransient(typeof(...))`. Expect IOC092.
7. **AddSingleton_TypeOf_ScopedService_EmitsIOC092**: `[Scoped]` but `AddSingleton(typeof(...))`. Expect IOC092.

`#region IOC094 - Open generic typeof()`
8. **OpenGeneric_TypeOf_EmitsIOC094**: `services.AddScoped(typeof(IRepository<>), typeof(Repository<>))`. Expect IOC094 at Info severity.

`#region ServiceDescriptor factory methods`
9. **ServiceDescriptor_Scoped_TypeOf_SameLifetime_EmitsIOC091**: `ServiceDescriptor.Scoped(typeof(IMyService), typeof(MyServiceImpl))` where `MyServiceImpl` is `[Scoped]`. Expect IOC091.
10. **ServiceDescriptor_Transient_TypeOf_ScopedService_EmitsIOC092**: `ServiceDescriptor.Transient(typeof(...))` where service is `[Scoped]`. Expect IOC092.

`#region No false positives`
11. **GenericTypeArgs_StillEmitIOC081**: Existing `services.AddScoped<IMyService, MyServiceImpl>()` (generic, not typeof) still emits IOC081 (regression check).
12. **TypeOf_SingleArg_NoInterface_NoFalsePositive**: `services.AddScoped(typeof(MyServiceImpl))` single-arg overload should be handled gracefully (no crash, may or may not emit diagnostic).

Each test uses inline source strings with the necessary `using` directives, IoCTools attribute imports, and a `Startup`-style class with `ConfigureServices(IServiceCollection services)`.

Follow the `SourceGeneratorTestHelper.CompileWithGenerator()` pattern from `ManualRegistrationOverlapTests.cs`. Assert on `result.Diagnostics` using FluentAssertions: `.Should().Contain(d => d.Id == "IOC091")` or `.Should().NotContain(d => d.Id == "IOC091")`.
